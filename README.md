# Android LLM API Client

Минимальное Android-приложение, которое демонстрирует работу с LLM через API.

Приложение отправляет текстовый запрос в языковую модель через HTTP API, получает ответ и отображает его в интерфейсе.

Проект был создан как учебное задание для понимания базового взаимодействия с LLM API.

---

# 🚀 Возможности

- отправка запроса в LLM через API  
- получение ответа от модели  
- отображение ответа в приложении  
- минимальная архитектура без лишней сложности  

---

# 🧠 Как это работает

Приложение выполняет следующий процесс:

1. Пользователь вводит текстовый запрос  
2. Приложение отправляет HTTP-запрос к LLM API  
3. API возвращает ответ модели  
4. Ответ отображается в интерфейсе приложения  

Схема взаимодействия:

```
User Input
    ↓
Android App
    ↓
LLM API (HTTP request)
    ↓
Model Response
    ↓
Android UI
```

# ⚙️ Запуск проекта

### 1. Клонировать репозиторий

```
git clone https://github.com/evgenyrsk/AiAdventChallenge.git
```

### 2. Открыть проект в Android Studio

```
File → Open → выбрать папку проекта
```

### 3. Добавить API ключ

В файле локальной конфигурации (local.properties) укажите свой API ключ:

```
AI_API_KEY=your_api_key
```

### 4. Запустить приложение

Запустите проект на:

- Android Emulator  
- физическом устройстве
