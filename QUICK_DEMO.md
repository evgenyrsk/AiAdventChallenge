# 🚀 Быстрая демонстрация Fitness MCP Tools (5 минут)

## 🎯 Что показываем
Естественный диалог в Android чате → автоматическое вызов fitness MCP tools

---

## 📋 Порядок действий

### 1. Запуск MCP Server (Терминал 1)
```bash
cd /Users/evgenyrsk/AndroidStudioProjects/AiAdventChallenge/mcp-server
gradle run
```
Ждём: `✅ MCP Server is running on http://10.0.2.2:8080`

---

### 2. Запуск Android приложения
В Android Studio: нажмите ▶️ (Run)
Или через эмулятор: откройте приложение

---

### 3. Демонстрация в чате

#### Сценарий 1: Добавление данных
**Напишите:** `Запиши мой вес 82.5 кг за сегодня, я сделал 10000 шагов`

**Что происходит:**
- ✅ FitnessRequestDetector детектирует запрос
- ✅ McpToolOrchestrator вызывает `add_fitness_log` MCP tool
- ✅ AI использует результат и отвечает пользователю

---

#### Сценарий 2: Получение сводки
**Напишите:** `Покажи мою сводку за неделю`

**Что происходит:**
- ✅ Детекция: GET_FITNESS_SUMMARY
- ✅ Вызов: `get_fitness_summary` MCP tool
- ✅ AI показывает агрегированные данные

---

#### Сценарий 3: Автоматическая сводка
**Напишите:** `Последняя сводка`

**Что происходит:**
- ✅ Детекция: GET_LATEST_SUMMARY
- ✅ Вызов: `get_latest_scheduled_summary` MCP tool
- ✅ AI показывает автоматически сгенерированную сводку

---
