## 🐛 Устранение проблем

### Проблема: "No tool found for this request"

**Причина:** FitnessRequestDetector не детектирует запрос

**Решение:**
- Проверьте логи: `adb logcat | grep FitnessRequestDetector`
- Убедитесь, что используете правильные ключевые слова

### Проблема: "Connection refused"

**Причина:** MCP Server не запущен

**Решение:**
- Проверьте, что сервер работает в Терминале 1
- Убедитесь, что URL правильный: `http://10.0.2.2:8080` (эмулятор)

### Проблема: "JSON parsing error"

**Причина:** JsonRpcModels не может распарсить ответ

**Решение:**
- Проверьте логи: `adb logcat | grep McpJsonRpcClient`
- Убедитесь, что все result types добавлены в `JsonRpcResult`

### Проблема: AI не использует результаты MCP tools

**Причина:** ToolExecutionResult не передаётся в LLM контекст

**Решение:**
- Проверьте логи: `adb logcat | grep McpToolOrchestrator`
- Убедитесь, что контекст формируется корректно
